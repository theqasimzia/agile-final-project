As a user, I need the ability to create a product in the catalog so that I can add new items to the system.
As a user, I need the ability to retrieve a product from the catalog so that I can view its details.
As a user, I need the ability to update a product in the catalog so that I can modify its information.
As a user, I need the ability to delete a product from the catalog so that I can remove unnecessary items.
As a user, I need the ability to "Like" a product in the catalog so that I can indicate my interest in it.
As a user, I need the ability to "Dislike" a product in the catalog so that I can express my dislike for it.
As a user, I need the ability to list all products in the catalog so that I can view an overview of available items.
As a user, I need the ability to query a subset of products in the catalog so that I can find specific items based on certain criteria.

As a stakeholder, I require automation to deploy new changes to the cloud to streamline the deployment process.
